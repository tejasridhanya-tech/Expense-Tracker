# Expense Tracker

## Mini Project

### Project Description

Expense Tracker is a simple Python application that helps users record their daily expenses and calculate the total amount spent.

The project is developed using Python and demonstrates the use of lists, loops, conditional statements, and calculations.

---

## Features

- Add new expenses
- View all expenses
- Display total spending
- Menu-driven program
- Easy to use

---

## Concepts Used

- Python Lists
- Loops
- Conditional Statements
- Functions
- Calculations
- User Input

---

## Software Used

- Python 3.10.11
- Thonny IDE

---

## How to Run

1. Open Thonny.
2. Open `expense_tracker.py`.
3. Click Run.
4. Select options from the menu.
5. Add expenses and calculate the total spending.
